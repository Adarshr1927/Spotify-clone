# Spotify Clone

A visually appealing Spotify Clone built using HTML and CSS. 
This project replicates the modern Spotify web interface with a responsive layout, stylish design, and music player inspired UI.

# Features
Modern Spotify-inspired user interface/n
Responsive sidebar and navigation menu
Music playlist and album sections
Stylish music player layout
Hover effects and smooth UI interactions
Fully responsive design for different screen sizes

# Technologies Used
HTML5 – Structure and layout
CSS3 – Styling, responsiveness, and animations

# Project Highlights
Clean and organized UI design
Responsive flexbox and grid layouts
Recreated Spotify homepage appearance
Custom styling for cards, buttons, and player controls

# Purpose
This project was created to practice:
Frontend web development
Responsive web design
CSS positioning and layouts
Flexbox and Grid concepts
UI cloning and styling techniques
